# Setup Guide

## Prerequisites

* Python 3.8+
* SerpAPI account & API key
* Internet connection

---

## Local Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/linkedin-profile-scraper-api.git
cd linkedin-profile-scraper-api
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Server

```bash
python app.py
```

### 4. Access API

```
http://127.0.0.1:10000/scrape
or
https://linkedin-scraper-api-atlc.onrender.com/scrape
```
